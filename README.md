# busylight_m365_linux

## What is `busylight_m365_linux`?

`busylight_m365_linux` is a lightweight tool that enables the use of Busylight devices on Linux systems by synchronizing their status with your Microsoft 365 presence.

## Installation

1. Download the latest `*none-any.whl` file from the releases and install it using `pipx`:

```
pipx install busylight_m365_linux-0.1.1-py3-none-any.whl
```

2. Install **BusyLight for Humans** by following the instructions here:

* [https://github.com/JnyJny/busylight](https://github.com/JnyJny/busylight)

  Make sure you install the version that includes the **Web API**.

3. Set up the required UDEV rules as documented in:

* [https://jnyjny.github.io/busylight/installation/#linux-udev-rules](https://jnyjny.github.io/busylight/installation/#linux-udev-rules)
